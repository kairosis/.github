# Support

## Getting help

### Documentation
The best place to start is the
[Kairosis documentation](https://github.com/Kairosis/docs).

### GitHub Discussions
For questions, ideas, and general conversation — open a discussion in
[Kairosis/kairosis/discussions](https://github.com/Kairosis/kairosis/discussions).

Use discussions for:
- Setup and configuration questions
- "How do I..." questions
- Connector authoring questions
- Sharing what you've built with Kairosis
- Ideas and feature requests you want to discuss before opening an issue

### GitHub Issues
For confirmed bugs only. Please search existing issues before opening a new one.

---

## Before asking for help

1. Check the [documentation](https://github.com/Kairosis/docs)
2. Search [existing discussions](https://github.com/Kairosis/kairosis/discussions)
3. Search [existing issues](https://github.com/Kairosis/kairosis/issues)
4. Check the [CHANGELOG](https://github.com/Kairosis/kairosis/blob/main/CHANGELOG.md)
   — your issue may already be fixed

---

## Security issues

Do not open public issues for security vulnerabilities.
See [SECURITY.md](./SECURITY.md) for how to report them privately.
